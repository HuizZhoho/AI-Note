# 万兆以太网 FPGA 术语表

```text
10GbE：
    10 Gigabit Ethernet，万兆以太网。

10GBASE-R：
    常见 10GbE PCS/PMA 体系，使用 64b/66b 编码。

64b/66b：
    每 64 bit 数据加 2 bit 同步头的编码方式。

AXI-Lite：
    常用于寄存器配置的轻量 AXI 接口。

AXI-Stream：
    FPGA 内部流式数据接口，用 tvalid/tready/tlast 表示数据和包边界。

BER：
    Bit Error Rate，误码率。

Block Lock：
    PCS 成功识别 66bit block 边界的状态。

CDC：
    Clock Domain Crossing，跨时钟域处理。

CDR：
    Clock Data Recovery，从串行数据中恢复时钟。

Descriptor：
    DMA 描述符，描述 buffer 地址、长度、状态和归属。

DMA：
    Direct Memory Access，硬件直接读写内存。

FCS：
    Frame Check Sequence，以太网帧校验。

Firmware：
    固件控制面，负责配置、状态监控、中断和 buffer 管理。

GT：
    Xilinx/AMD FPGA 高速串行收发器。

IFG：
    Inter Frame Gap，以太网帧间隔。

ILA：
    Integrated Logic Analyzer，FPGA 内部逻辑分析仪。

Jumbo Frame：
    超过标准 1500 MTU 的大帧，常见 9000 bytes 左右。

Local Fault：
    本地检测到链路故障。

MAC：
    Media Access Control，处理以太网帧层。

MM2S：
    Memory Mapped to Stream，DMA 从内存读出转为流。

PFC：
    Priority Flow Control，按优先级流控。

PCS：
    Physical Coding Sublayer，物理编码子层。

PMA：
    Physical Medium Attachment，物理介质附着子层。

PMD：
    Physical Medium Dependent，物理介质相关子层。

PRBS：
    Pseudo Random Binary Sequence，伪随机序列，用于链路测试。

QPLL/CPLL：
    GT 中的 PLL 资源，用于高速串行时钟。

Remote Fault：
    对端报告链路故障。

RX_LOS：
    Receive Loss of Signal，SFP+ 接收无光信号。

S2MM：
    Stream to Memory Mapped，DMA 从流写入内存。

SFP+：
    常见 10GbE 光模块/线缆接口。

tdata：
    AXI-Stream 数据。

tkeep：
    AXI-Stream 字节有效标记。

tlast：
    AXI-Stream 包结束标记。

tready：
    AXI-Stream 接收方准备好。

tuser：
    AXI-Stream 用户侧边带信号，常用于错误标记。

tvalid：
    AXI-Stream 发送方数据有效。

XGMII：
    10G MAC 和 PHY/PCS 之间的逻辑接口。

Zero-copy：
    零拷贝，减少 CPU 数据搬运以提升性能。
```
