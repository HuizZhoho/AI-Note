# 《The All-New Switch Book》博客与网易云音乐播客发布包

本目录把 LAN Switching 的核心知识整理成 8 篇博客稿和 8 期网易云音乐播客脚本，面向网络入门者。内容为原创学习整理，不是原书翻译，也不复述原书段落；结构参考《The All-New Switch Book》的主题脉络，并按中文发布场景重新组织。

## 发布定位

- 博客用途：公众号、知乎、CSDN、个人博客等图文平台。
- 网易云音乐用途：以“播客/声音节目”形式发布，不按音乐作品发布。
- 读者/听众：零基础到初级网络运维、嵌入式以太网学习者、准备系统学习交换机的人。
- 系列口吻：少堆标准编号，多解释“为什么需要它、它怎么工作、出问题怎么看”。
