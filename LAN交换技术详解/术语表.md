# LAN Switching 术语表

## A

```text
Access Port：
    接终端的端口，通常只属于一个 VLAN，发给终端的帧通常不带 tag。

ACL：
    Access Control List，访问控制列表，用于匹配和过滤流量。

Aging：
    MAC 表项老化机制，长期不用的表项会被删除。
```

## B

```text
BPDU：
    Bridge Protocol Data Unit，STP/RSTP/MSTP 用于交换拓扑信息的控制报文。

Bridge：
    网桥，透明桥是现代二层交换机的理论基础。

Broadcast：
    广播，目的 MAC 为 ff:ff:ff:ff:ff:ff。

Broadcast Domain：
    广播域，广播帧能到达的二层范围。
```

## C

```text
CAM：
    Content Addressable Memory，内容寻址存储器，常用于 MAC 表高速查找。

CoS：
    Class of Service，服务类别，常和 802.1p 优先级相关。

CRC：
    循环冗余校验，用于检测帧错误。

Cut-through：
    交换机收到目的地址后就开始转发，延迟低但错误帧可能被转发。
```

## D

```text
Designated Port：
    STP 中某个网段负责转发的端口。

Duplex Mismatch：
    双工不匹配，一端全双工一端半双工，常导致性能差和错误包。
```

## E

```text
Edge Port：
    RSTP 中连接终端的边缘端口，可快速转发。

EtherType：
    以太网帧字段，指示上层协议类型。
```

## F

```text
FCS：
    Frame Check Sequence，以太网帧校验序列。

Filtering：
    过滤，交换机判断帧不需要转发到某端口。

Flooding：
    泛洪，交换机把帧发到同 VLAN 内除入口外的多个端口。

Forwarding：
    转发，交换机把帧发送到正确出口。
```

## G

```text
GARP：
    Generic Attribute Registration Protocol，通用属性注册协议。

GMRP：
    GARP Multicast Registration Protocol，组播注册协议。

GVRP：
    GARP VLAN Registration Protocol，VLAN 注册协议。
```

## H

```text
HOL Blocking：
    Head-of-Line Blocking，队头阻塞。
```

## I

```text
IGMP Snooping：
    交换机监听 IGMP 报文，建立 IPv4 组播转发表。

Ingress：
    入方向，帧进入交换机的方向。
```

## L

```text
LACP：
    Link Aggregation Control Protocol，链路聚合控制协议。

Learning：
    学习，交换机根据源 MAC 记录 MAC 到端口的对应关系。

Link Aggregation：
    链路聚合，多条物理链路组成一条逻辑链路。

LLDP：
    Link Layer Discovery Protocol，链路层发现协议。
```

## M

```text
MAC Address Table：
    MAC 地址表，记录 VLAN、MAC、端口之间的对应关系。

MAC Flooding：
    通过大量伪造源 MAC 填满交换机 MAC 表的攻击。

MSTP：
    Multiple Spanning Tree Protocol，多生成树协议。

MTU：
    Maximum Transmission Unit，最大传输单元。
```

## P

```text
PAUSE：
    IEEE 802.3x 流控帧，让对端暂停发送一段时间。

PCP：
    Priority Code Point，802.1Q tag 中的 3 bit 优先级字段。

PVID：
    Port VLAN ID，端口默认 VLAN，用于未打 tag 的入方向帧。
```

## Q

```text
QoS：
    Quality of Service，服务质量，通过分类、标记、排队、调度等方式保障关键业务。
```

## R

```text
Root Bridge：
    STP 生成树中的根桥。

Root Port：
    非根交换机到根桥最佳路径上的端口。

RSTP：
    Rapid Spanning Tree Protocol，快速生成树协议。
```

## S

```text
SNMP：
    Simple Network Management Protocol，网络管理协议。

STP：
    Spanning Tree Protocol，生成树协议，用于防止二层环路。

Store-and-forward：
    交换机完整接收并校验帧后再转发。

Switch Fabric：
    交换结构，连接输入端口和输出端口的内部转发结构。
```

## T

```text
TCAM：
    Ternary Content Addressable Memory，三态内容寻址存储器，支持 0/1/任意匹配。

Trunk Port：
    承载多个 VLAN 的端口，通常用于交换机之间或交换机到路由器/服务器。
```

## V

```text
VID：
    VLAN ID。

VLAN：
    Virtual LAN，虚拟局域网，用于划分二层广播域。

VLAN Hopping：
    VLAN 跳跃攻击，攻击者试图进入不属于自己的 VLAN。

VOQ：
    Virtual Output Queue，虚拟输出队列，用于缓解队头阻塞。
```
