# 以太网芯片术语表

ARP：
    IPv4 地址解析协议，用 IP 找 MAC。

AXI-Stream：
    FPGA 内部常用流接口，通过 tdata、tkeep、tvalid、tready、tlast 表示数据和包边界。

BAR：
    PCIe Base Address Register，CPU 通过它映射 NIC 寄存器空间。

BMCR：
    Basic Mode Control Register，PHY 基本控制寄存器。

BMSR：
    Basic Mode Status Register，PHY 基本状态寄存器。

CPU Port：
    交换芯片连接管理 CPU 的端口。

Descriptor：
    DMA 描述符，描述 buffer 地址、长度、状态和控制位。

Descriptor Ring：
    驱动或 Firmware 与硬件共享的一圈 DMA 描述符队列。

Doorbell：
    驱动写给硬件的通知寄存器，用来告诉 NIC/DMA 有新的 descriptor 可处理。

DMA：
    Direct Memory Access，硬件直接读写内存。

DSA：
    Linux Distributed Switch Architecture，把交换芯片端口建模为 Linux 网络接口。

FCS：
    Frame Check Sequence，以太网帧尾 CRC 校验。

GMII：
    Gigabit Media Independent Interface，千兆 MAC-PHY 并行接口。

ILA：
    Integrated Logic Analyzer，FPGA 内部逻辑抓取工具。

LACP：
    Link Aggregation Control Protocol，链路聚合控制协议。

MAC：
    Media Access Control，以太网帧收发控制层。

MDI：
    Medium Dependent Interface，PHY 面向网线或磁性器件的一侧。

MDIO/MDC：
    PHY 管理接口和管理时钟。

MSI-X：
    PCIe 消息中断机制，常用于多队列 NIC。

Modbus TCP：
    工业以太网常见应用协议，运行在 TCP 502 端口，用于寄存器读写和设备控制。

NAPI：
    Linux 网络驱动中断加轮询机制。

NIC：
    Network Interface Card，网卡或网卡控制器。

PCS：
    Physical Coding Sublayer，物理编码子层。

PHY：
    Physical Layer Transceiver，以太网物理层收发器芯片。

PMA：
    Physical Medium Attachment，物理媒介连接子层。

PMD：
    Physical Medium Dependent，物理媒介相关子层。

PVID：
    Port VLAN ID，端口接收 untagged 帧时归属的 VLAN。

RGMII：
    Reduced Gigabit Media Independent Interface，常见千兆 MAC-PHY 接口。

RMII：
    Reduced Media Independent Interface，常见 10/100M MAC-PHY 接口。

RSS：
    Receive Side Scaling，网卡按 hash 把流量分到多 RX 队列。

SGMII：
    Serial Gigabit Media Independent Interface，串行千兆 MAC-PHY 接口。

SFP+：
    常见 10GbE 小型可插拔光模块接口，也可使用 DAC 线缆。

Switch ASIC：
    交换芯片，用硬件流水线完成二层/三层转发。

TCAM：
    Ternary CAM，支持 0/1/通配匹配，常用于 ACL。

TSO：
    TCP Segmentation Offload，网卡硬件 TCP 分段。

VLAN：
    Virtual LAN，二层逻辑隔离机制。

XGMII：
    10GbE MAC 和 PCS 之间的逻辑接口。
